Asteroids - Python Sample 1   Asteroids image

Astroids game code - a nice showcase of using classes in python 2.7. Game can be played from your browser window of your computer. You do not need to load python software to play. Game does not work on smartphones or iPads. It also does NOT work on Internet Explorer.

From your desktop computer, please use Chrome, Firefox, or Safari browsers. Click on the link below to view the Python code and to play. The link will open a CodeSkulptor window as a pop-up to your browser. Please allow pop-ups. Click the run button on top-left to start (see image below). Enjoy!
